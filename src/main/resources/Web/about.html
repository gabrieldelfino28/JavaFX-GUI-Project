# JavaFX-GUI-Project ☕
## Description - (Eng-US) Welcome. This is a project that I was thinking of, where I combine all the exercises from batches I had to solve on college during my second semester with Java, into one big project. It's basically the same algorithms and logic part from the Java-Algorithms-Project but with a new JavaFX UI and WebView in the app. This project involves: * Arrays * Loops * Functions and Methods * Recursive * Objects * Abstract classes and more ## Descrição - (Pt-BR)
